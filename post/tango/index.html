もくもく英単語

もくもく英単語とは?

もくもくと三択英単語問題を解くアプリです。
鬱陶しい演出などはありません。
問題の文字を触ると問題を読み上げます。
※英語版では日本語辞書を登録すると読み上げます。
猫を触ると答えの候補を読み上げます。
それではひたすら問題を解きましょう！

JP

How to use.

This is an application for solving three-choice English word problems.
There are no annoying effects.
If you touch the letters of the question, the question will be read out loud.
In the English version, if you register a Japanese dictionary, it will read the question out loud.
When you touch the cat, it will read out the possible answers.
So let’s solve the problems!

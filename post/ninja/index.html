録音忍者

録音忍者とは?

最速(当社比)で録音が開始できます。
アプリを起動するだけで録音が開始されます。
NOタッチで録音です。
録音感度が設定できます。
手裏剣が回っていると録音されています。
手裏剣をタッチすると録音を停止し、設定画面へ移動します。
手裏剣は隠すことが可能です。
録音済データは全てファイルに保存されます。
データは日付ごとに分けられます。
データは録音開始時間毎に作られます。
著作権は守りましょう。 違法な利用、差別的な利用を禁止します。

JP

You can record.

You can start recording in the fastest time (our comparison).
Just launch the app to start recording.
NO touch to record.
You can set the recording sensitivity.
When the shuriken is spinning, the recording is being made.
Touch the shuriken to stop recording and go to the settings screen.
The shuriken can be hidden.
All recorded data will be saved in a file. Data is divided by date.
Data is created for each recording start time.
Illegal or discriminatory use is prohibited.
